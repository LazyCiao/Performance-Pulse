Financial Pulse

A Django web app that reflects my passion for building data-driven solutions. It empowers users to search for public companies and provides comprehensive financial analysis.

By leveraging the Alpha Vantage API, Financial Pulse offers real-time financial data, Matplotlib charts, and valuable insights on company performance as well as the latest news articles.

The app showcases my expertise in working with external APIs, data manipulation, and utilizing libraries NLP, Matplotlib and Numpy. I'm proud to present this project as a demonstration of my ability to create efficient and effective applications.

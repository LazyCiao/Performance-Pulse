Equity Tracker

A Django web app that reflects my passion for building data-driven solutions. It empowers users to search for public companies and provides comprehensive financial analysis.

By leveraging the Alpha Vantage API, Financial Pulse offers real-time financial data, Matplotlib charts, and valuable insights on company performance as well as the latest news articles.

The app showcases my expertise in working with external APIs, data manipulation, and utilizing libraries NLP, Matplotlib and Numpy. I'm proud to present this project as a demonstration of my ability to create efficient and effective applications.

Crypto Market

A dynamic Django web application dedicated to the world of cryptocurrencies. This project demonstrates my proficiency in web development and data visualization.

Through interactions with various cryptocurrency APIs, the app enables users to explore top cryptocurrencies, visualize their price movements using interactive candlestick charts powered by Plotly, and delve into detailed information for each cryptocurrency.

By integrating APIs, creating responsive UIs, and utilizing libraries like Plotly, Matplotlib, and requests, this app reflects my skills in fetching real-time data, generating insightful visualizations, and building user-friendly interfaces.
